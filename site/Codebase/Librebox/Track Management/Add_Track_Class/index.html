    String _extension_hint_for_native_path_by_magic(native_path_string: String) 🔗

    Infers mp3/wav/ogg by reading the first bytes from the native picker URI. This works even when the native path has no extension (e.g. …/document/msf%3A1000000088). Returns “” when the type cannot be determined.
