    Constants

    ROOT_MUSIC_DIR = "res://Music/Song Metadatas/" 🔗

    Song metadata resource used by LibreBox and DJ_Controller. Stores music metadata plus stream/waveform references for built-in and user-imported tracks. Root directory for built-in song metadata resources.

    USER_SONG_METADATA_DIR = "user://Music/" 🔗

    User-imported songs (Add Track on device) save .tres here so they persist on Quest/Android.

    Property Descriptions

    StringName Song_Title 🔗

    Song title shown in UI.

    Artist Main_Artist 🔗

    Primary artist for this song.

    Array[Artist] Guest_Artists 🔗

    Optional list of featured artists.

    Album Song_Album 🔗

    Album resource for this song. Singles can still use a same-name album resource.

    EGenre.E_ID3Genres[] Song_Genres 🔗

    Genre IDs describing this song. Done in Ints, use Int_to_Id3Genre_str() to get the Genre Display

    AudioStream Audio_File 🔗

    Playable audio stream asset for this song.

    String Audio_File_Path 🔗

    For user-imported songs (Add Local): path to the file under user:// so it persists. When set, get_audio_stream() loads from this path if Audio_File is null.
